Форматы фалов.

.DOC или .doc (от англ. document ) — расширение имени файла, используемое для файлов,
представляющих текст, с разметкой или без. Расширение .DOC часто использовалось для
обозначения текстовых файлов, без форматирования, однако позже стало
использоваться для двоичных форматов с разметкой.

Portable Document Format FDF) — межплатформенный открытый формат электронных документов,
разработанный фирмой Adobe Systems в 1992 году с использованием ряда возможностей языка
PostScript.
Документ FDF

ZIP — Популярный формат сжатия данных и архивации файлов. Файл в этом формате
обычно имеют расширение zip и хранят в сжатом или несжатом виде один или несколько
файлов, которое можно из него извлечь путём распаковки с помощью специальной
программы.
